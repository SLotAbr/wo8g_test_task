# wo8g_test_task
API организационной структуры, техническое задание на позицию "Python Backend Developer". Текст задания доступен [прямо в репозитории](https://github.com/SLotAbr/wo8g_test_task/blob/main/Backend%20Python%20Task.pdf).

## Детали
Приложение работает с деревом департаментов и их сотрудниками. Основной сложностью задания является не интеграция дополнительных технологий, а реализация проверок для пограничных случаев и вспомогательных функций для взаимодействия с деревьями, таких как: обход дерева вширь с использованием очереди, проверка наличия циклов при модификации дерева, сериализация дерева в JSON формат.

Стек:
* SQLAlchemy, PostgreSQL, Alembic
* Docker, Docker Compose
* FastAPI, Pydantic, Pytest
* Все взаимодействия асинхронны

## Инструкции по установке и запуску
1. Сервис целиком обёрнут в Docker контейнеры. Для первого запуска: `make apply-first-migrations-and-run`
1. Помимо ручного тестирования, доступно автоматическое: `make test`

Swagger UI документация с описанием всех методов приложения доступна локально: `http://localhost:8000/docs`
