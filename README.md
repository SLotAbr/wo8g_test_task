# wo8g_test_task
API организационной структуры. Пока файл содержит минимум для соответствия запросам задания. Если интересны детали, то возвращайтесь позже.

## Инструкции по установке и запуску
1. Сервис целиком обёрнут в Docker контейнеры. Для первого запуска: `make apply-first-migrations-and-run`
1. Помимо ручного тестирования, доступно автоматическое: `make test`

Swagger UI документация с описанием всех методов приложения доступна локально: `http://localhost:8000/docs`
